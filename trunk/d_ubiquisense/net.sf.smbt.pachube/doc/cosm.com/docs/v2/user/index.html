Users

The users API is designed to allow third party applications to be built upon the Cosm API. It enables developers to create, modify, manage and delete users. Created users 'belong' to the developer that created them in the sense that the creating account is essentially an admin for the new user. The attributes available via the API are:

Attribute Description Required? Settable
login The login name for the user (i.e. their Cosm ID) Yes Yes
full name The full name of the user No Yes
email The email address for the user No Yes
website The user's website No Yes
about Brief biography or description of the user No Yes
organisation The organisation that this user belongs to No Yes
api_key The master API key for this user No No
deliver_email Setting that defines whether this user wants to receive messages sent by Cosm's internal message system in their email inbox. Note this setting doesn't affect any admin notices sent in relation to your account No Yes
display_activity Whether the user's activity information should be displayed on their profile page No Yes
display_information Whether or not the user's profile info should be displayed on their profile page No Yes
display_stats Whether or not the user's stats should be displayed on their profile page No Yes
receive_forum_notifications Whether or not the user wishes to receive forum notifications No Yes
subscribed_to_mailings Whether or not the user wishes to be subscribed to email newsletters sent by Cosm No Yes
time_zone The user's time zone. This can either be set using one of the values listed here: http://api.cosm.com/#displaying-times-in-different-zones-via-the-api or by specifying the time offset as an numeric value.    
roles A list of roles the user currently has or should have when you are creating new users. A role is what defines the capabilities of the user. Note that in order to use this API to create users you must pre-arrange with Cosm so that your account can be given a list of roles that it is allowed to create Yes Yes
creatable_roles List of roles that you are allowed to use when creating new users. These will have been pre-arranged in agreement with Cosm; so if blank you won't be able to create any users. Please contact us if that is something you require No No

Note, that in order to create users via the API your account has to have this functionality enabled, but you can view and update your own details at any time. If you'd like the capability to create users, then please contact us and let us know what you are doing, and we should be able to enable it for your account.
