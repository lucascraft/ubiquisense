#jPachube Documentation

Introduction¶

This provides method for accessing Pachube's Api allowing the user to Create/Retrive/Update and Delete Feeds/Datastreams and Triggers. The API provides a simple method of creating feeds without any knowledge of EEML but also provides methods to provide eeml to pachube manually.The API can generate the eeml for the requested action.

The source code is available at this location:

http://code.google.com/p/jpachube/source/browse/#svn/trunk/src/Pachube

The Documentation are available at this location:

http://jpachube.googlecode.com/svn/trunk/doc/index.html

Error handling is basic at this time, providing only reports of failed HTTP commands, the status of an executed command can be retrieved from the header items of a HttpResponse object using the key "Status".

This brief example shows how the api can be used to create a Feed.

In this example the Feed object 'f' is not live, the p.createFeed() method returns a live feed object which can be used to update the online feed. 

import Pachube.Data;
import Pachube.Feed;
import Pachube.FeedFactory;
import Pachube.Pachube;
import Pachube.PachubeException;


public class CreateFeed {

public static void main(String arsg[]) throws InterruptedException{
        try {
                    //Creates a Pachube object authenicated using the provided API KEY

                    Pachube p = new Pachube(API KEY HERE);
                    Feed f = new Feed();
                    f.setTitle("JAVA API TEST FEED");
                    Data a = new Data();
                    a.setId(0);
                    a.setMaxValue(100d);
                    a.setMinValue(0d);
                    a.setTag("Test");
                    a.setValue(30d);
                    f.addData(a);
                    Feed g = p.createFeed(f);
                    //The Feed 'f' is does not represent the feed on pachube any
                    // Changes made to this object will not alter the online feed.
                    System.out.println("The id of the new feed is:");
                    System.out.println(g.getId());

            } catch (PachubeException e) {
                //If an exception occurs it will print the error message from the failed 
                // HTTP command
                System.out.println(e.errorMessage);
            }
        }

}

To update an existing feed's datastream the example code is shown below; This code will update the feed with id 2993 

try {
            Pachube p = new Pachube(API KEY HERE);
            Feed f = p.getFeed(2993);
            f.updateDatastream(0, 90d);
    } catch (PachubeException e) {
            System.out.println(e.errorMessage);
    }

In this API Triggers are modeled using the Trigger class which is capable of forming the correct HTTP headers. The code below create a Trigger on the feed 2993 datastream 0 and will post to the specified URL if the streams value is greater than 10. 

try {
                        Pachube p = new Pachube(APU_KEY);
                        Trigger t = new Trigger();
                        t.setEnv_id(2993);
                        t.setStream_id(0);
                        t.setThreshold(10d);
                        t.setType(TriggerType.gt);

                        t.setUrl(new URL("http://google.com"));

                        p.createTrigger(t);
                } catch (PachubeException e) {
                        // TODO Auto-generated catch block
                        e.printStackTrace();
                } catch (MalformedURLException e) {
                        // TODO Auto-generated catch block
                        e.printStackTrace();
                }
