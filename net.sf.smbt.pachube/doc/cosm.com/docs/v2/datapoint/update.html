Updating a datapoint

Example http://api.cosm.com/v2/feeds/1977/datastreams/1/datapoints/2010-07-28T07:48:22.014326Z
Formats json, xml, csv
Methods POST
Returns HTTP headers only

A PUT request will modify the value of the datapoint for this timestamp.

Examples

JSON

{
  "value":"297"
}

XML

<eeml xmlns="http://www.eeml.org/xsd/0.5.1" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" version="0.5.1" xsi:schemaLocation="http://www.eeml.org/xsd/0.5.1 http://www.eeml.org/xsd/0.5.1/0.5.1.xsd">
    <environment>
        <data>
            <datapoints>
                <value>11</value>
            </datapoints>
        </data>
    </environment>
</eeml>

CSV

297
