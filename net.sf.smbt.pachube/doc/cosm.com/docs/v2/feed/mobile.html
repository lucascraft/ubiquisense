Mobile feeds

Example http://api.cosm.com/v2/feeds/3819.json?duration=1day
Formats xml, json
Methods GET
Returns feed with waypoint data

The Cosm API supports mobile feeds and will store the history of changes to the locatin of a feed automatically. To make this happen, the disposition of the feed needs setting to 'mobile' and then when you change the location it will be saved into the history. To retrieve the location history, simply make a normal history query and you will see a 'waypoints' element in the location for the feed with a list of points in it. Any time one of the coordinates changes, all of the points are stored so that they stay in sync.
